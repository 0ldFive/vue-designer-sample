### 功能演示

本项目展示了如何将打印设计器集成到 **Vue 3 + Element Plus** 应用中，包含以下特性：

1. **主题切换**：支持深色/浅色模式切换，设计器自动跟随。
2. **国际化 (i18n)**：支持中英文切换，设计器语言自动跟随。
3. **路由集成**：在 Vue Router 中使用设计器。
4. **业务页面打印**：在非设计器页面（如表单页）直接调用打印和导出功能，演示了变量注入与模板联动。
5. **打印参数调试**：提供可视化的打印参数调试面板，支持切换本地/远程打印模式、调整打印机选项等。

### 快速开始

1. **克隆项目**
   ```bash
   git clone https://github.com/0ldFive/vue-designer-sample.git
   cd vue-designer-sample
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **启动开发服务器**
   ```bash
   npm run dev
   ```
